# Frequently Asked Questions (FAQ)

Common questions from servers and managers using MagiDesk POS.

---

## General Questions

### Q: What is MagiDesk POS?

**A:** MagiDesk POS is a point-of-sale system designed for bars and billiard halls. It helps you manage tables, take orders, process payments, and track inventory.

### Q: Do I need internet to use it?

**A:** Yes, the system requires internet connection to work. If internet is down, use manual backup procedures and enter data later.

### Q: Can I use it on my phone?

**A:** Currently, MagiDesk POS is a desktop application for Windows computers. Mobile version may be available in the future.

### Q: What if I make a mistake?

**A:** Most mistakes can be corrected:
- Orders can be modified before confirming
- Items can be removed from orders
- Payments can be refunded (with manager approval)
- Always ask your manager if unsure

## Login Questions

### Q: I forgot my password. What do I do?

**A:** Ask your manager to reset your password. They can generate a new temporary password for you.

### Q: Why can't I log in?

**A:** Common reasons:
- Wrong username or password

## Table Questions

### Q: When should I start a table session?

**A:** Start a session immediately when customers sit down at a table. This starts the timer and allows you to take orders.

### Q: What if I forget to start a session?

**A:** Start it as soon as you remember. The timer will start from that point. You may need to manually adjust the time cost if customers were there longer.

### Q: Can I start multiple sessions on the same table?

**A:** No, each table can only have one active session at a time. You must stop the current session before starting a new one.

### Q: What if customers want to move to a different table?

**A:** You can use the "Move Table" feature, or stop the current session and start a new one on the new table. Add the existing order items to the new table.

### Q: How do I know how long customers have been at a table?

**A:** When you click on an occupied table, you'll see the duration (e.g., "45 minutes"). This helps estimate the time cost.

## Order Questions

### Q: How do I add items to an order?

**A:** Click on the occupied table, then click "Add Items". Browse the menu, select items, enter quantities, and click "Confirm Order".

### Q: Can I modify an order after confirming it?

**A:** Yes, you can:
- Add more items
- Remove items (if not yet prepared)
- Change quantities
- Cancel items (with kitchen/bar approval)

### Q: What if an item is out of stock?

**A:** The system will show the item as "Out of Stock" or grayed out. You cannot add out-of-stock items. Suggest alternatives to the customer.

### Q: How do I know when an order is ready?

**A:** Check the order status:
- **Pending** (blue) - Order received
- **Preparing** (yellow) - Kitchen/bar is working on it
- **Ready** (green) - Item is ready for delivery

### Q: What if I add the wrong item?

**A:** If the item hasn't been prepared yet, you can remove it from the order. If it's already being prepared, check with kitchen/bar first.

## Payment Questions

### Q: How do I process a payment?

**A:** 
1. Stop the table session
2. Review the bill
3. Click "Process Payment"
4. Select payment method
5. Enter amount
6. Click "Process Payment"

### Q: What if the payment amount doesn't match the bill?

**A:** The system will show an error. Verify:
- Payment amount equals bill total
- For split payments, all amounts add up to total
- No typos in amount entry

### Q: Can I process a partial payment?

**A:** Yes, but this is usually for special situations. The bill will show as "Partially Paid" and you'll need to process the remaining amount later.

### Q: What if a customer wants to split the bill?

**A:** Use the "Split Payment" feature:
1. Select "Split Payment"
2. Enter amount for each payment method
3. Verify total equals bill amount
4. Process payment

### Q: How do I apply a discount?

**A:** 
1. In payment screen, click "Apply Discount"
2. Enter discount amount or percentage
3. Enter reason
4. Manager approval may be required
5. Apply discount and process payment

### Q: What if a payment fails?

**A:** 
- Check error message
- Verify payment amount
- Try different payment method
- Wait a moment and try again
- Contact manager if issue persists

## Refund Questions

### Q: How do I process a refund?

**A:** 
1. Go to "Payments" page
2. Find the payment to refund
3. Click "Refund"
4. Enter refund amount
5. Select refund method
6. Enter reason
7. Process refund

**Note:** Manager approval is usually required for refunds.

### Q: Can I refund part of a payment?

**A:** Yes, you can process partial refunds. Enter the amount you want to refund (must be less than or equal to original payment).

### Q: What if a customer paid with card but wants cash refund?

**A:** You can select "Cash" as the refund method, even if original payment was card. Always get manager approval for this.

## Receipt Questions

### Q: How do I print a receipt?

**A:** Receipts print automatically after payment. If you need to print again, click "Print Receipt" button.

### Q: What if the receipt won't print?

**A:** 
- Check printer is on and has paper
- Check printer is connected
- Try printing manually
- Contact manager if issue persists

### Q: Can I email a receipt to a customer?

**A:** If this feature is enabled, click "Email Receipt" after payment and enter the customer's email address.

## Inventory Questions (Managers)

### Q: How do I check if an item is in stock?

**A:** Go to "Inventory" page and find the item. You'll see the current quantity. Items show as:
- **In Stock** (green) - Above threshold
- **Low Stock** (yellow) - Below threshold
- **Out of Stock** (red) - Zero quantity

### Q: How do I update stock when a delivery arrives?

**A:** 
1. Go to "Inventory"
2. Find each item
3. Click "Adjust Stock"
4. Enter new quantity or adjustment amount
5. Select "Delivery received" as reason
6. Save

### Q: What is a reorder threshold?

**A:** The minimum quantity before you should reorder. When stock falls below this number, the system alerts you.

## Menu Questions (Managers)

### Q: How do I change a price?

**A:** 
1. Go to "Menu Management"
2. Find the item
3. Click "Edit"
4. Change the price
5. Enter reason (optional)
6. Save

### Q: How do I mark an item as out of stock?

**A:** 
1. Go to "Menu Management"
2. Find the item
3. Click "Edit"
4. Toggle "Available" to "No"
5. Save

The item will disappear from the menu for servers.

## Report Questions (Managers)

### Q: How do I view today's sales?

**A:** 
1. Go to "Reports"
2. Click "Sales Reports"
3. Select "Today"
4. View the report

### Q: Can I export reports?

**A:** Yes! Click "Export" on any report and choose format (PDF, Excel, or CSV).

### Q: What reports are available?

**A:** 
- Sales reports (daily, weekly, monthly)
- Inventory reports
- Customer reports
- Payment reports
- Item reports

## Troubleshooting Questions

### Q: The system is slow. What should I do?

**A:** 
- Check internet connection
- Close other applications
- Refresh the page
- Log out and log back in
- Contact IT support if issue persists

### Q: I see an error message. What does it mean?

**A:** 
- Read the error message carefully
- It usually tells you what's wrong
- Try the suggested fix
- Ask your manager if unsure
- Contact IT support if it's a technical error

### Q: The printer isn't working. What do I do?

**A:** 
- Check printer is on
- Check printer has paper
- Check printer is connected
- Try printing a test page
- Contact manager or IT support

### Q: I can't find a menu item. Where is it?

**A:** 
- Item might be out of stock (check inventory)
- Item might have been removed from menu
- Try searching for the item name
- Check different categories
- Ask manager if item should be available

## Training Questions

### Q: I'm new. Where do I start?

**A:** 
1. Read the [Server Quick Start Guide](./server-quick-start.md)
2. Practice with a test table
3. Shadow an experienced server
4. Ask questions
5. Review the [Server Training Manual](./server-training-manual.md)

### Q: How long does training take?

**A:** 
- Quick start: 10 minutes
- Basic training: 1-2 hours
- Full training: 2-3 hours
- Comfortable use: 1-2 shifts of practice

### Q: Can I practice without affecting real data?

**A:** Ask your manager if there's a test mode or practice environment. Otherwise, practice during slow periods with real tables (but be careful!).
