## Using Shortcuts

### How to Use

**To use a shortcut:**
1. Press and hold the modifier key (Ctrl, Shift, etc.)
2. Press the other key
3. Release both keys
4. Action executes

**Example:**
- To go to Dashboard: Hold `Ctrl`, press `D`, release both

### When Shortcuts Work

**Shortcuts work when:**
- You're on the correct page
- No dialog is open (some shortcuts)
- Field is not focused (some shortcuts)
- System supports the shortcut

**Shortcuts don't work when:**
- Typing in a text field
- Dialog is open (may close dialog instead)
- System doesn't support the shortcut

## Finding Available Shortcuts

### Check System

**To find available shortcuts:**
1. Look for tooltips (hover over buttons)
2. Check menu items (may show shortcuts)
3. Ask your manager
4. Contact IT support
5. Check system documentation

### Common Patterns

**Most systems use:**
- `Ctrl + [Letter]` for navigation
- `Esc` for cancel/close
- `Enter` for confirm/submit
- `Tab` for navigation between fields

## Tips

### ✅ Best Practices

- Learn common shortcuts you use often
- Use shortcuts to work faster
- Don't rely on shortcuts (mouse works too)
- Ask if shortcut doesn't work

### ❌ Common Mistakes

- Trying shortcuts that don't exist
- Using shortcuts in wrong context
- Forgetting shortcuts exist
- Not learning shortcuts
