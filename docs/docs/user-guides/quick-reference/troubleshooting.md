# Troubleshooting Guide for Users

This guide helps you solve common problems when using MagiDesk POS.

---

## Login Problems

### Can't Log In

**Symptoms:**
- "Invalid username or password" error
- Login button doesn't work
- System won't accept credentials

**Solutions:**
1. **Check credentials:**
   - Verify username is correct
   - Verify password is correct
   - Check Caps Lock is off

2. **Try typing password in notepad first:**
   - Type password in Notepad
   - Copy and paste into login field
   - Ensures no typos

3. **Reset password:**
   - Ask manager to reset password
   - Use new temporary password
   - Change password on first login

4. **Check system status:**
   - Ask if system is down
   - Check if others can log in
   - Contact IT support if needed

## Table Problems

### Table Won't Start

**Symptoms:**
- "Start Session" button doesn't work
- Error when trying to start
- Table stays green

**Solutions:**
1. **Check table status:**
   - Is table really available?
   - Is another session active?
   - Refresh the page

2. **Try again:**
   - Wait a moment
   - Try starting again
   - Check for error messages

3. **Contact manager:**
   - If issue persists
   - May need system check
   - May need to free table manually

### Table Won't Stop

**Symptoms:**
- "Stop Session" button doesn't work
- Error when trying to stop
- Table stays red

**Solutions:**
1. **Check session exists:**
   - Verify session is active
   - Check table details
   - Refresh the page

2. **Try again:**
   - Wait a moment
   - Try stopping again
   - Check for error messages

3. **Contact manager:**
   - If issue persists
   - May need manual intervention
   - May need system check

### Wrong Table Information

**Symptoms:**
- Wrong server name
- Wrong duration
- Wrong order total

**Solutions:**
1. **Verify information:**
   - Check table details
   - Compare with reality
   - Note discrepancies

2. **Fix if possible:**
   - Stop and restart session (if early)
   - Contact manager to correct
   - Document the issue

3. **Report to manager:**
   - Explain the problem
   - Provide details
   - Get help fixing

## Order Problems

### Can't Add Items

**Symptoms:**
- Items don't appear in menu
- "Add" button doesn't work
- Error when adding items

**Solutions:**
1. **Check item availability:**
   - Is item out of stock?
   - Is item in menu?
   - Try searching for item

2. **Check session:**
   - Is table session started?
   - Is table occupied?
   - Refresh the page

3. **Try again:**
   - Wait a moment
   - Try adding again
   - Check for error messages

4. **Contact manager:**
   - If item should be available
   - If issue persists
   - May need menu update

### Items Not Showing in Order

**Symptoms:**
- Added items but they don't appear
- Order list is empty
- Items disappeared

**Solutions:**
1. **Refresh order view:**
   - Click "View Order" again
   - Refresh the page
   - Check order list

2. **Verify items were added:**
   - Check if order was confirmed
   - Review order history
   - Check for error messages

3. **Re-add items:**
   - Add items again
   - Verify order is confirmed
   - Check order list

4. **Contact manager:**
   - If items are lost
   - If issue persists
   - May need system check

### Order Status Not Updating

**Symptoms:**
- Status stays "Pending"
- Status doesn't change
- Can't see when items are ready

**Solutions:**
1. **Refresh the page:**
   - Click refresh button
   - Reload the page
   - Check status again

2. **Check with kitchen/bar:**
   - Ask about item status
   - Verify items are ready
   - Update manually if needed

3. **Wait a moment:**
   - Status may update automatically
   - System may be processing
   - Check again in a minute

## Payment Problems

### Payment Won't Process

**Symptoms:**
- "Process Payment" button doesn't work
- Error when processing
- Payment fails

**Solutions:**
1. **Check payment details:**
   - Verify amount equals bill total
   - Check payment method is selected
   - Verify all required fields

2. **Check system:**
   - Is system running?
   - Is internet connected?
   - Are others having issues?

3. **Try again:**
   - Wait a moment
   - Try processing again
   - Check for error messages

4. **Contact manager:**
   - If issue persists
   - May need system check
   - May need IT support

### Payment Amount Wrong

**Symptoms:**
- Payment amount doesn't match bill
- Can't process payment
- Error about amount mismatch

**Solutions:**
1. **Verify bill total:**
   - Check bill calculation
   - Review time cost
   - Review items cost
   - Verify total

2. **Check payment amount:**
   - Ensure amount equals bill total
   - For split payments, verify all add up
   - Check for typos

3. **Fix and retry:**
   - Correct the amount
   - Verify total matches
   - Process payment again

### Receipt Won't Print

**Symptoms:**
- Receipt doesn't print
- Printer error
- Blank receipt

**Solutions:**
1. **Check printer:**
   - Is printer on?
   - Does printer have paper?
   - Is printer connected?

2. **Check printer selection:**
   - Go to Settings → Printer
   - Verify correct printer selected
   - Test print

3. **Try manual print:**
   - Click "Print Receipt" button
   - Select printer
   - Try printing

4. **Contact manager:**
   - If printer issue persists
   - May need IT support
   - Can email receipt as backup

## System Problems

### System is Slow

**Symptoms:**
- Pages take long to load
- Buttons respond slowly
- System feels sluggish

**Solutions:**
1. **Check internet:**
   - Is internet connection good?
   - Try other websites
   - Check network

2. **Close other applications:**
   - Close unnecessary programs
   - Free up computer resources
   - Restart computer if needed

3. **Refresh the page:**
   - Click refresh button
   - Reload the page
   - May fix temporary issues

4. **Contact IT support:**
   - If issue persists
   - May need system check
   - May need performance optimization

### System is Frozen

**Symptoms:**
- Nothing responds
- Screen is frozen
- Can't click anything

**Solutions:**
1. **Wait a moment:**
   - System may be processing
   - Give it 30 seconds
   - Don't click multiple times

2. **Refresh the page:**
   - Press F5 to refresh
   - Reload the page
   - May restore functionality

3. **Restart application:**
   - Close the application
   - Reopen it
   - Log back in

4. **Contact IT support:**
   - If system is completely frozen
   - May need system restart
   - May need technical help

### Error Messages

**Symptoms:**
- Error popup appears
- Error message on screen
- System shows error

**Solutions:**
1. **Read the error:**
   - Read error message carefully
   - Note what it says
   - Understand what went wrong

2. **Try suggested fix:**
   - Error may suggest solution
   - Follow instructions
   - Try the fix

3. **Common errors:**
   - "Network error" → Check internet
   - "Server error" → Wait and retry
   - "Validation error" → Check your input
   - "Permission denied" → Ask manager

4. **Contact manager:**
   - If error persists
   - If you don't understand error
   - If error prevents work

## Getting Help

### When to Ask for Help

**Ask your manager for:**
- Login issues
- Permission problems
- Policy questions
- Discount approvals
- Refund approvals

**Ask IT support for:**
- System errors
- Technical problems
- Printer issues
- Network problems
- Application crashes

### How to Report Issues

**When reporting, include:**
- What you were trying to do
- What happened instead
- Error message (if any)
- When it occurred
- Steps you took
- Your name

**Example:**
- "I was trying to process a payment for Table 5. When I clicked 'Process Payment', I got an error saying 'Amount mismatch'. The bill was $50.00 and I entered $50.00. This happened at 2:30 PM. I tried twice. - John Smith"
