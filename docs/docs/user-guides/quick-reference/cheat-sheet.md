## Starting a Table Session

```
1. Click on available table (green)
2. Click "Start Session"
3. Select your name
4. Click "Start"
```

**Timer starts automatically!**

---

## Taking an Order

```
1. Click on occupied table (red)
2. Click "Add Items"
3. Select items from menu
4. Enter quantities
5. Click "Add to Order"
6. Click "Confirm Order"
```

---

## Processing Payment

```
1. Click on occupied table
2. Click "Stop Session"
3. Review bill
4. Click "Process Payment"
5. Select payment method
6. Enter amount
7. Add tip (optional)
8. Click "Process Payment"
```

**Receipt prints automatically!**

---

## Split Payment

```
1. In payment screen, select "Split Payment"
2. Enter amount for each method:
   - Card: $50
   - Cash: $30
   - Mobile: $20
3. Verify total = bill amount
4. Click "Process Split Payment"
```

---

## Applying Discount

```
1. In payment screen, click "Apply Discount"
2. Enter discount amount or %
3. Enter reason
4. Click "Apply"
5. Process payment
```

**Manager approval may be required!**

## Common Issues - Quick Fix

| Problem | Quick Fix |
|---------|-----------|
| Can't log in | Check username/password, ask manager to reset |
| Table won't start | Check if already occupied, refresh page |
| Payment failed | Verify amount, check payment method, try again |
