# Viewing Sales Reports (Managers)

This guide explains how managers view and understand sales reports.

---

## Overview

**Sales reports show:**
- Total sales
- Number of transactions
- Average order value
- Payment methods breakdown
- Top selling items
- Hourly/daily trends

---

## Daily Sales Report

### Step-by-Step

1. **Click "Reports"** in navigation
2. **Click "Sales Reports"**
3. **Select "Today"** from date range
4. **View report:**
   - Total sales for today
   - Number of transactions
   - Average order value
   - Payment methods breakdown
   - Top selling items
   - Hourly sales breakdown

### Understanding the Report

**Total Sales:**
- Sum of all payments today
- Shows revenue
- Compare to targets

**Transactions:**
- Number of orders/payments
- Shows volume
- Helps understand traffic

**Average Order Value:**
- Total sales ÷ Transactions
- Shows customer spending
- Helps with pricing

**Payment Methods:**
- Cash: $X
- Card: $Y
- Mobile: $Z
- Shows payment preferences

**Top Items:**
- Best selling items today
- Quantities sold
- Revenue per item

---

## Weekly/Monthly Reports

### Viewing Longer Periods

1. **Click "Reports"**
2. **Click "Sales Reports"**
3. **Select date range:**
   - This week
   - This month
   - Custom range
4. **View trends:**
   - Sales trends over time
   - Day-by-day comparison
   - Peak hours analysis
   - Seasonal patterns

---

## Exporting Reports

### Step-by-Step

1. **View any report**
2. **Click "Export"** button
3. **Select format:**
   - PDF (for printing)
   - Excel (for analysis)
   - CSV (for data import)
4. **Click "Download"**
5. **File downloads** to your computer

---

## Understanding Metrics

### Key Metrics Explained

**Total Sales:**
- How much money you made
- Primary revenue metric
- Compare to targets

**Average Order Value:**
- Average amount per order
- Helps understand customer spending
- Useful for pricing decisions

**Transactions:**
- Number of orders
- Shows business volume
- Helps with staffing

**Top Items:**
- What's selling best
- Helps with inventory
- Guides menu decisions

---

## Best Practices

### ✅ Do's

- Review reports daily
- Compare to previous periods
- Identify trends
- Use for decision-making
- Export for records

### ❌ Don'ts

- Ignore reports
- Make decisions without data
- Skip regular reviews
- Ignore anomalies
