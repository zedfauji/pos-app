# Checking Daily Totals (Managers)

This guide explains how managers check daily sales totals and key metrics.

---

## Quick Daily Check

### On Dashboard

**When you log in, dashboard shows:**
- **Today's Sales:** Total revenue for today
- **Active Tables:** How many tables are occupied
- **Pending Orders:** Orders waiting to be prepared
- **Recent Payments:** Latest transactions

**Quick overview** of how the day is going

---

## Detailed Daily Totals

### Step-by-Step

1. **Click "Reports"**
2. **Click "Sales Reports"**
3. **Select "Today"**
4. **View daily totals:**
   - Total sales
   - Number of transactions
   - Average order value
   - Payment methods
   - Top items

---

## Understanding Daily Totals

### Total Sales

**What it means:**
- Sum of all payments today
- Your revenue for the day
- Primary metric

**What to check:**
- Compare to target
- Compare to yesterday
- Compare to same day last week

### Number of Transactions

**What it means:**
- How many orders/payments today
- Business volume
- Customer count (approximately)

**What to check:**
- Compare to average
- Check if busy or slow
- Helps with staffing

**What it means:**
- Average amount per order

**What to check:**
- Is it increasing?
- Compare to target
- Helps with pricing

## End of Day Review

### Daily Closing

**Before closing, review:**
