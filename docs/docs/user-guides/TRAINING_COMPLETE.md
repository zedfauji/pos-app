## Training Program Structure

### Day 1: Basics (2-3 hours)

**Morning Session (1.5 hours):**
- Introduction to MagiDesk POS
- Logging in and interface overview
- Starting and stopping table sessions
- Basic order taking

**Afternoon Session (1.5 hours):**
- Practice with test scenarios
- Processing payments
- Handling common situations
- Q&A session

### Day 2: Advanced Features (2-3 hours)

**Morning Session (1.5 hours):**
- Split payments
- Discounts and refunds
- Order modifications
- Troubleshooting

**Afternoon Session (1.5 hours):**
- Real-world practice
- Shadow experienced server
- Handle actual customers (with supervision)
- Review and feedback

### Day 3: Independent Practice

- Work independently with support available
- Practice all procedures
- Ask questions as needed
- Build confidence

## How to Use These Guides

### For New Employee Onboarding

1. **Day 1:**
   - Give them [Server Quick Start Guide](./server-quick-start.md)
   - Walk through login and basic interface
   - Practice starting/stopping tables
   - Practice taking a simple order

2. **Day 2:**
   - Review [Server Training Manual](./server-training-manual.md)
   - Practice all procedures
   - Handle real customers with supervision
   - Answer questions

3. **Day 3:**
   - Independent practice
   - Reference [Server Daily Tasks](./server-daily-tasks.md)
   - Use [Cheat Sheet](./quick-reference/cheat-sheet.md) as needed
   - Check [FAQ](./quick-reference/faq.md) for questions

### For Manager Training

1. **Session 1:**
   - Review [Manager Quick Start Guide](./manager-quick-start.md)
   - Walk through manager dashboard
   - Practice viewing reports
   - Practice managing menu

2. **Session 2:**
   - Review [Manager Training Manual](./manager-training-manual.md)
   - Practice all manager tasks
   - Review [Manager Daily Tasks](./manager-daily-tasks.md)
   - Answer questions
