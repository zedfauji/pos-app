# Manager Quick Start Guide

**Time to Complete:** 15 minutes  
**For:** New managers learning the system

Welcome! This guide will get you up and running with MagiDesk POS management features quickly.

---

## Step 1: Logging In (2 minutes)

### Manager Login

1. **Open MagiDesk App**
2. **Enter your manager username**
3. **Enter your password**
4. **Click "Login"**

**Manager Access:**
- You'll see additional menu items
- You have access to reports and settings
- You can manage users and permissions

## Step 3: Viewing Reports (3 minutes)

### Daily Sales Report

**Step-by-Step:**

1. **Click "Reports"** in navigation
2. **Click "Sales Reports"**
3. **Select date range** (today, this week, this month)
4. **View report:**
   - Total sales
   - Number of transactions
   - Average order value
   - Payment methods breakdown

### Inventory Report

**Step-by-Step:**

1. **Click "Reports"**
2. **Click "Inventory Reports"**
3. **View:**
   - Current stock levels
   - Low stock items
   - Items sold today
   - Reorder recommendations

---

## Step 4: Managing Menu Items (3 minutes)

### Updating Prices

**Step-by-Step:**

1. **Click "Menu Management"**
2. **Find the item** to update
3. **Click "Edit"**
4. **Change the price**
5. **Click "Save"**

**Important:**
- Price changes take effect immediately
- Old prices are saved in history
- Customers see new prices right away

### Marking Items as Out of Stock

**Step-by-Step:**

1. **Click "Menu Management"**
2. **Find the item**
3. **Click "Edit"**
4. **Toggle "Available" to "No"**
5. **Click "Save"**

**What happens:**
- Item disappears from menu for servers
- Item shows as "Out of Stock" if servers try to add it
- You can mark it available again when stock arrives

---

## Step 5: Managing Inventory (2 minutes)

### Checking Stock Levels

**Step-by-Step:**

1. **Click "Inventory"**
2. **View current stock:**
   - Item name
   - Current quantity
   - Low stock alerts (red)
   - Reorder threshold

### Updating Stock

**Step-by-Step:**

1. **Click "Inventory"**
2. **Find the item**
3. **Click "Adjust Stock"**
4. **Enter new quantity** or adjustment amount
5. **Enter reason** (e.g., "Delivery received")
6. **Click "Save"**

---

## Step 6: Processing Refunds (2 minutes)

### When to Process Refunds

**Common scenarios:**
- Customer cancellation
- Wrong item delivered
- Customer dissatisfaction
- Manager-approved refunds

**Step-by-Step:**

1. **Click "Payments"**
2. **Find the payment** to refund
3. **Click "Refund"**
4. **Enter refund amount** (full or partial)
5. **Select refund method**
6. **Enter reason**
7. **Click "Process Refund"**

**Manager Approval:**
- You can approve refunds
- Large refunds may need owner approval
- All refunds are logged for audit

## Common Manager Tasks

### Task 1: End of Day Review

**What to do:**
1. View daily sales report
2. Review all transactions
3. Check for any issues
4. Verify cash matches system
5. Generate end-of-day report

### Task 2: Approving Discounts

**When server requests discount:**
1. Review discount request
2. Verify it's legitimate
3. Approve or deny
4. Discount is applied to bill

### Task 3: Managing Staff

**Adding a new server:**
1. Click "Users"
2. Click "Add User"
3. Enter server details
4. Assign role (Server)
5. Set initial password
6. User can log in

**Resetting password:**
1. Click "Users"
2. Find the user
3. Click "Reset Password"
4. New temporary password is generated
5. Give password to user

### Task 4: Handling Issues

**If server reports problem:**
1. Listen to the issue
2. Check system status
3. Try to reproduce the issue
4. Fix if possible
5. Escalate to IT if needed
6. Document the issue
