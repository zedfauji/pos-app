# Modifying Orders

This guide explains how to modify orders after they've been created.

---

## When to Modify Orders

**Modify orders when:**
- Customer wants to change something
- Wrong item was added
- Quantity needs to change
- Item needs to be removed
- Customer changes their mind

**You can modify:**
- Before items are prepared
- After items are prepared (with kitchen/bar approval)
- At any time during the session

---

## Types of Modifications

### Adding Items
- Add more items to existing order
- See: [Adding Items to Orders](./adding-items.md)

### Removing Items
- Remove items from order
- Cancel items that haven't been prepared

### Changing Quantities
- Increase or decrease quantities
- Update item counts

### Cancelling Items
- Cancel items entirely
- Remove from order completely

---

## Step-by-Step: Removing Items

### Step 1: Open the Order

**Click on the occupied table**

**Click "View Order" or "Order Details"**

**You'll see:**
- List of all items
- Quantities
- Prices
- Status of each item

### Step 2: Find the Item

**Locate the item** you want to remove

**Items show:**
- Item name
- Quantity
- Price
- Status (Pending, Preparing, Ready, Delivered)

### Step 3: Remove the Item

**Click "Remove" or "Cancel" button** on the item

**Or:**
- Right-click on item
- Select "Remove" from menu

### Step 4: Confirm Removal

**System asks for confirmation:**
- "Are you sure you want to remove this item?"
- Shows item details
- Shows impact on total

**Click "Confirm" or "Yes"**

**What happens:**
- ✅ Item is removed from order
- ✅ Order total updates
- ✅ Item is cancelled
- ✅ Kitchen/bar is notified (if item was being prepared)

---

## Step-by-Step: Changing Quantities

### Step 1: Open the Order

**Click on the table**

**Click "View Order"**

### Step 2: Find the Item

**Locate the item** you want to modify

### Step 3: Edit Quantity

**Click "Edit" button** on the item

**Or:**
- Click on quantity number
- Enter new quantity

### Step 4: Enter New Quantity

**Enter the new quantity:**
- Increase: Enter higher number
- Decrease: Enter lower number
- Remove: Enter 0 (removes item)

**Example:**
- Current: 2 beers
- Change to: 3 beers
- Enter "3"

### Step 5: Save Changes

**Click "Save" or "Update"**

**What happens:**
- ✅ Quantity is updated
- ✅ Order total recalculates
- ✅ Kitchen/bar is notified (if needed)
- ✅ Changes are saved

---

## Important Notes

### ⚠️ Before Modifying

- **Check item status** (can it be modified?)
- **Verify with customer** what they want
- **Check with kitchen/bar** if item is being prepared
- **Understand impact** on order total

### Item Status and Modifications

**Pending Items:**
- ✅ Can be removed easily
- ✅ Can change quantity
- ✅ No problem modifying

**Preparing Items:**
- ⚠️ May need kitchen/bar approval
- ⚠️ Check before removing
- ⚠️ May not be cancellable

**Ready/Delivered Items:**
- ❌ Usually cannot be removed
- ❌ May need refund instead
- ❌ Check with manager

### ✅ Best Practices

- Modify orders promptly
- Verify changes with customer
- Check item status before modifying
- Get approval if item is being prepared
- Update order accurately

### ❌ Common Mistakes

- Removing items that are ready
- Not checking with kitchen/bar
- Making changes without verifying
- Forgetting to save changes
- Rushing and making errors

---

## Special Situations

### Customer Wants to Cancel Entire Order

**If customer wants to cancel everything:**
1. Check if any items are ready/delivered
2. If none ready, cancel all items
3. If some ready, may need refund
4. Contact manager if needed

### Item is Being Prepared

**If item is already being prepared:**
- Check with kitchen/bar first
- May not be able to cancel
- May need to proceed with item
- Customer may need to pay anyway

### Wrong Item Was Added

**If wrong item was added:**
- Remove wrong item
- Add correct item
- Verify order is correct
- Confirm with customer

---

## Troubleshooting

### Problem: "Cannot Remove Item"

**Possible causes:**
- Item is already delivered
- Item is being prepared
- System restriction

**Solutions:**
1. Check item status
2. Check with kitchen/bar
3. May need refund instead
4. Contact manager if needed

### Problem: "Changes Not Saving"

**What to do:**
- Verify you clicked "Save"
- Check for error messages
- Try again
- Refresh page if needed

### Problem: "Order Total Wrong After Modification"

**What to check:**
- Verify all changes were saved
- Check quantities are correct
- Review order list
- Recalculate if needed
