# Viewing Order Status

This guide explains how to check the status of orders and items.

---

## Why Check Order Status?

**Checking order status helps you:**
- Know when items are ready
- Coordinate with kitchen/bar
- Keep customers informed
- Manage delivery timing
- Track order progress

---

## How to View Order Status

### Step 1: Open the Table

**Click on the occupied table** (red table)

**You'll see:**
- Table details
- Current order information
- "View Order" button

### Step 2: Open Order Details

**Click "View Order" or "Order Details"**

**Order details screen shows:**
- Order number
- Table number
- Server name
- Created time
- List of all items
- Status of each item
- Order total

## Understanding Statuses

### Pending

**What it means:**
- Order has been received
- Waiting to be prepared
- Kitchen/bar hasn't started yet

**What to do:**
- Wait for status to change
- Check back in a few minutes
- Don't rush kitchen/bar

### Preparing

**What it means:**
- Kitchen/bar is working on it
- Item is being made
- Should be ready soon

**What to do:**
- Wait for it to be ready
- Prepare to deliver when ready
- Don't remove item (it's being made)

### Ready

**What it means:**
- Item is finished
- Ready to be delivered
- Waiting for you to pick up

**What to do:**
- Pick up item from kitchen/bar
- Deliver to customer
- Mark as delivered

### Delivered

**What it means:**
- Item has been delivered
- Customer has received it
- Part of completed order

**What to do:**
- Nothing - it's done!
- Check if customer needs anything else

### Cancelled

**What it means:**
- Item was removed from order
- Won't be prepared
- Customer won't receive it

**What to do:**
- Verify cancellation was intentional
- Check if customer still wants it
- May need to re-add if mistake

## Order-Level Status

### Complete Order Status

**The entire order also has a status:**

**Pending:**
- Some items not yet started
- Order is being worked on
- Not all items ready

**Partial:**
- Some items ready, some not
- Can deliver ready items
- Wait for remaining items

**Ready:**
- All items are ready
- Can deliver everything
- Order is complete

**Delivered:**
- All items delivered
- Order is complete
- Ready for payment

## Checking Status Regularly

### During Service

**Check order status:**
- Every few minutes
- Before going to kitchen/bar
- When customer asks
- Before delivering items

**Why it matters:**
- Knows when to pick up items
- Keeps customers informed
- Manages timing
- Prevents items sitting too long

### Best Practices

‚úÖ **Do:**
- Check status regularly
- Pick up items when ready
- Deliver promptly
- Mark items as delivered
- Keep customers informed

‚ùå **Don't:**
- Ignore order status

## Marking Items as Delivered

### When Item is Ready

**After you deliver an item:**

1. **Open order details**
2. **Find the item** that's ready
3. **Click "Mark as Delivered"**
4. **Item status changes** to "Delivered"

**Why this matters:**
- Tracks what's been served
- Helps with accurate billing
- Shows kitchen/bar what's done
- Updates order status

---

## Troubleshooting

### Problem: "Status Not Updating"

**Possible causes:**
- Kitchen/bar hasn't updated
- System delay
- Network issue

**Solutions:**
1. Wait a moment and refresh
2. Check with kitchen/bar
3. Refresh the page
4. Check network connection

### Problem: "Item Shows Ready But Isn't"

**What to do:**
- Check with kitchen/bar
- Verify item is actually ready
- Don't deliver if not ready
- Update status if needed

### Problem: "Can't See Order Status"

**What to do:**
- Refresh the page
- Check you're on right table
- Try viewing order again
- Contact manager if issue persists
