# Adding Items to Orders

This guide explains how to add items to an existing order.

---

## When to Add Items

**Add items when:**
- Customers want to order more
- Customers forgot something
- Customers want additional items
- Order needs to be updated

**You can add items:**
- At any time during the session
- Before or after items are delivered
- Multiple times if needed

---

## Step-by-Step Instructions

### Step 1: Open the Table

**Click on the occupied table** (red table)

**You'll see:**
- Table details
- Current order (if any)
- "Add Items" button

### Step 2: Open Add Items

**Click "Add Items" button**

**Menu appears:**
- Categories on left or top
- Search bar
- Item cards/grid

### Step 3: Find the Item

**Option 1: Browse Categories**
- Click category (Food, Drinks, Snacks, etc.)
- Scroll through items
- Find what you want

**Option 2: Search**
- Type item name in search bar
- Results appear as you type
- Click on the item

### Step 4: Select the Item

**Click on the item** you want to add

**Item details appear:**
- Item name
- Description
- Price
- Availability
- Modifiers (if available)

### Step 5: Configure the Item

**Enter quantity:**
- Default is 1
- Change if customer wants more
- Example: 2 beers → Enter "2"

**Select modifiers (if available):**
- Size: Small, Medium, Large
- Toppings: Add/remove options
- Customizations: Special requests

### Step 6: Add to Order

**Click "Add to Order" or "Add to Cart"**

**What happens:**
- Item is added to cart
- Cart shows at bottom/side
- You can see:
  - Items in cart
  - Quantities
  - Prices
  - Running total

### Step 7: Add More Items (Optional)

**Continue adding:**
- Browse or search for more items
- Select and add to cart
- Repeat until order is complete

**You can add:**
- Multiple items at once
- Different quantities
- Items with different modifiers

### Step 8: Review Cart

**Before confirming, check:**
- ✅ All items are correct
- ✅ Quantities are right
- ✅ Modifiers are correct
- ✅ Total looks reasonable

**Cart shows:**
- List of all items
- Quantity for each
- Price for each
- Line total
- **Cart total**

### Step 9: Confirm Order

**Click "Confirm Order" or "Submit"**

**What happens:**
- ✅ New items are added to existing order
- ✅ Order is updated
- ✅ Items are sent to kitchen/bar
- ✅ Inventory is checked
- ✅ Stock is deducted

**You'll see:**
- Confirmation message
- Updated order total
- New items appear in order list

---

## Adding to Existing Order

### If Order Already Has Items

**When you add items to existing order:**
- New items are added to the list
- Order total updates automatically
- All items stay together
- One order, multiple items

**Example:**
- Existing order: 2 beers ($11.00)
- Add: 1 pizza ($12.99)
- **New total: $23.99**

### Order Updates Automatically

**The system:**
- Combines all items
- Calculates new total
- Updates order status
- Sends new items to kitchen/bar

---

## Important Notes

### ⚠️ Before Adding

- **Check item availability** (is it in stock?)
- **Verify quantity** with customer
- **Check modifiers** are correct
- **Review cart** before confirming

### ✅ Best Practices

- Add items accurately
- Verify quantities
- Check availability before adding
- Review cart before confirming
- Confirm with customer (optional)

### ❌ Common Mistakes

- Adding wrong items
- Wrong quantities
- Forgetting modifiers
- Not checking availability
- Rushing and making errors

---

## Special Situations

### Item is Out of Stock

**If item shows as unavailable:**
- Item will be grayed out
- Cannot add it
- Suggest alternatives
- Inform customer

### Customer Changes Mind

**Before confirming:**
- Can remove items from cart
- Can change quantities
- Can cancel and start over

**After confirming:**
- May need to cancel items
- Check with kitchen/bar first
- Some items may not be cancellable

### Adding Large Quantities

**For large orders:**
- Add items in batches if needed
- Review cart periodically
- Confirm when complete
- System handles large orders fine

---

## Troubleshooting

### Problem: "Item Not Available"

**What it means:** Item is out of stock

**Solution:**
- Cannot add unavailable items
- Suggest alternatives
- Inform customer
- Check inventory if you have access

### Problem: "Can't Add Item"

**Possible causes:**
- Item is out of stock
- System error
- Item was removed

**Solutions:**
1. Check item availability
2. Try again
3. Search for item
4. Ask manager if item should be available

### Problem: "Item Added But Not Showing"

**What to do:**
- Refresh the order view
- Check order list
- Verify item was added
- Try adding again if needed
