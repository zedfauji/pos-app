# Employee Onboarding Checklist

Use this checklist to ensure new employees are properly trained on MagiDesk POS.

---

## Pre-Training Setup

### Before Training Begins

- [ ] **Create user account** for new employee
- [ ] **Set initial password** (temporary)
- [ ] **Assign appropriate role** (Server, Cashier, Manager, etc.)
- [ ] **Verify system access** works
- [ ] **Prepare training materials** (print guides if needed)
- [ ] **Set up practice environment** (test tables, test data)
- [ ] **Schedule training time** (2-3 hours recommended)

---

## Day 1: Basic Training

### Morning Session (1.5 hours)

- [ ] **Introduction** (15 min)
  - [ ] Welcome and overview
  - [ ] System purpose and benefits
  - [ ] What they'll learn today

- [ ] **Login and Navigation** (30 min)
  - [ ] First time login
  - [ ] Password change
  - [ ] Interface overview
  - [ ] Navigation basics
  - [ ] **Practice:** Login and navigate

- [ ] **Table Management** (45 min)
  - [ ] Starting a table session
  - [ ] Understanding table status
  - [ ] Stopping a table session
  - [ ] **Practice:** Start and stop test tables

### Afternoon Session (1.5 hours)

- [ ] **Order Taking** (45 min)
  - [ ] Creating an order
  - [ ] Adding items to orders
  - [ ] Viewing order status
  - [ ] **Practice:** Take a complete order

- [ ] **Payment Processing** (45 min)
  - [ ] Processing a payment
  - [ ] Understanding the bill
  - [ ] Payment methods
  - [ ] Printing receipts
  - [ ] **Practice:** Process a test payment

### End of Day 1

- [ ] **Review** what was learned
- [ ] **Answer questions**
- [ ] **Assign reading:** Server Quick Start Guide
- [ ] **Schedule Day 2**

---

## Day 2: Advanced Training

### Morning Session (1.5 hours)

- [ ] **Split Payments** (30 min)
  - [ ] How split payments work
  - [ ] Processing split payments
  - [ ] **Practice:** Split payment scenarios

- [ ] **Discounts and Refunds** (45 min)
  - [ ] Applying discounts (with approval)
  - [ ] Processing refunds (with approval)
  - [ ] When to get approval
  - [ ] **Practice:** Discount and refund scenarios

- [ ] **Order Modifications** (15 min)
  - [ ] Adding items to existing orders
  - [ ] Removing items
  - [ ] Changing quantities

### Afternoon Session (1.5 hours)

- [ ] **Real-World Practice** (60 min)
  - [ ] Shadow experienced server
  - [ ] Handle real customers (with supervision)
  - [ ] Practice all procedures
  - [ ] Get feedback

- [ ] **Troubleshooting** (30 min)
  - [ ] Common issues
  - [ ] How to solve problems
  - [ ] When to ask for help
  - [ ] Review FAQ

### End of Day 2

- [ ] **Review** advanced features
- [ ] **Answer questions**
- [ ] **Assign reading:** Server Training Manual
- [ ] **Schedule Day 3**

---

## Day 3: Independent Practice

### Full Shift Practice

- [ ] **Work independently** with support available
- [ ] **Practice all procedures:**
  - [ ] Starting/stopping tables
  - [ ] Taking orders
  - [ ] Processing payments
  - [ ] Handling special situations
- [ ] **Ask questions** as needed
- [ ] **Get feedback** from manager
- [ ] **Build confidence**

### End of Day 3

- [ ] **Final review** with manager
- [ ] **Assessment** of skills
- [ ] **Address any concerns**
- [ ] **Confirm readiness** for independent work

---

## Skills Assessment

### After Training, Employee Should Be Able To:

**For Servers:**
- [ ] Log in successfully
- [ ] Navigate the system
- [ ] Start and stop table sessions
- [ ] Take orders accurately
- [ ] Process payments correctly
- [ ] Handle split payments
- [ ] Apply discounts (with approval)
- [ ] Process refunds (with approval)
- [ ] Print receipts
- [ ] Troubleshoot common issues

**For Managers:**
- [ ] All server skills, plus:
- [ ] View and understand reports
- [ ] Manage menu items
- [ ] Manage inventory
- [ ] Manage users
- [ ] Process refunds
- [ ] Approve discounts
- [ ] Configure settings

## Post-Training Support

### First Week

- [ ] **Check in daily** with new employee
- [ ] **Answer questions** promptly
- [ ] **Provide support** as needed
- [ ] **Review performance**
- [ ] **Address any issues**

### Ongoing

- [ ] **Regular check-ins**
- [ ] **Additional training** if needed
- [ ] **Update training** as system changes
- [ ] **Gather feedback** for improvements
