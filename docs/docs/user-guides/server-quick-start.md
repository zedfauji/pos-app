# Server Quick Start Guide

**Time to Complete:** 10 minutes  
**For:** New servers learning the basics

Welcome! This guide will get you up and running with MagiDesk POS in just 10 minutes.

## Step 1: Logging In (2 minutes)

### How to Log In

1. **Open the MagiDesk App** on your computer
2. **Enter your username** (provided by your manager)
3. **Enter your password** (you'll set this on first login)
4. **Click "Login"**

**What you'll see:**
- Main dashboard with tables
- Navigation menu on the left
- Your name in the top right corner

**Troubleshooting:**
- **Forgot password?** Ask your manager to reset it
- **Can't log in?** Check your username and password are correct
- **App won't open?** Contact IT support

## Step 3: Starting a Table Session (3 minutes)

### When a Customer Sits Down

**Step-by-Step:**

1. **Click on an available table** (green table)
2. **Click "Start Session"** button
3. **Select your name** from the server list
4. **Click "Start"**

**What happens:**
- Table turns red (occupied)
- Timer starts counting
- System creates a session
- You can now take orders

**Important Notes:**
- Always start a session when customers sit down
- Timer calculates the bill automatically
- You can add items to the session

---

## Step 4: Taking an Order (3 minutes)

### Adding Items to a Table

**Step-by-Step:**

1. **Click on the occupied table** (red table)
2. **Click "Add Items"** or "View Order"
3. **Browse the menu** (food, drinks, etc.)
4. **Click on items** to add them
5. **Enter quantity** if needed
6. **Click "Add to Order"**
7. **Click "Confirm"**

**What happens:**
- Items are added to the order
- Order is sent to kitchen/bar
- Items appear in the order list
- Total updates automatically

**Tips:**
- You can add multiple items at once
- You can modify orders before confirming
- Check inventory availability (items show as "Out of Stock" if unavailable)

---

## Step 5: Processing Payment (3 minutes)

### When Customer is Ready to Pay

**Step-by-Step:**

1. **Click on the occupied table**
2. **Click "Stop Session"** or "Bill"
3. **Review the bill:**
   - Time cost (based on how long they stayed)
   - Items cost (what they ordered)
   - **Total amount**
4. **Click "Process Payment"**
5. **Select payment method:**
   - Cash
   - Card
   - Mobile Payment
   - Split Payment (multiple methods)
6. **Enter payment amount**
7. **Click "Process"**

**What happens:**
- Payment is recorded
- Bill status changes to "Paid"
- Receipt is generated
- Table becomes available (green)

**Important:**
- Always verify the total before processing
- For split payments, ensure amounts add up correctly
- Receipt prints automatically (if printer is connected)

## Common Scenarios

### Scenario 1: Customer Wants to Add Items

**What to do:**
1. Click on the table
2. Click "Add Items"
3. Select new items
4. Confirm order
5. Items are added to existing order

### Scenario 2: Customer Wants to Cancel an Item

**What to do:**
1. Click on the table
2. Click "View Order"
3. Find the item to cancel
4. Click "Remove" or "Cancel"
5. Confirm cancellation

### Scenario 3: Customer Wants to Split the Bill

**What to do:**
1. When processing payment, select "Split Payment"
2. Enter amounts for each payment method
3. Example: $50 card, $30 cash
4. Ensure total equals bill amount
5. Process payment

### Scenario 4: Customer Wants a Discount

**What to do:**
1. When processing payment, click "Apply Discount"
2. Enter discount amount or percentage
3. Enter reason (optional)
4. Discount is applied
5. Process payment with discounted amount
