# Printing Receipts

This guide explains how to print receipts for customers.

---

## When Receipts Print

**Receipts print automatically:**
- After payment is processed
- When payment is successful
- System generates receipt automatically

**You can also print manually:**
- If receipt didn't print
- If customer needs another copy
- For records

---

## Automatic Printing

### After Payment

**When you process a payment:**

1. Payment is processed
2. Receipt is generated automatically
3. Receipt prints to default printer
4. Receipt is ready for customer

**What you need:**
- Printer must be on
- Printer must have paper
- Printer must be connected
- Printer must be selected in settings

---

## Manual Printing

### Step 1: After Payment

**After payment is processed:**

**Receipt screen appears with:**
- Receipt preview
- "Print Receipt" button
- "Email Receipt" button (if enabled)
- Receipt details

### Step 2: Print Receipt

**Click "Print Receipt" button**

**What happens:**
- Receipt is sent to printer
- Printer prints the receipt
- Receipt is ready

**If multiple printers:**
- Select printer from list
- Choose the correct printer
- Click "Print"

### Step 3: Give Receipt to Customer

**Hand receipt to customer:**
- Verify receipt printed correctly
- Check all information is visible
- Thank customer

**Receipt includes:**
- **Business Information:**
  - Business name

  - Phone number

- **Receipt Information:**
  - Receipt number
  - Date and time
  - Table number

  - List of all items
  - Quantity for each
  - Price for each

- **Financial Summary:**
  - Subtotal
  - Time cost (if applicable)
  - Discount (if any)
  - Tax (if applicable)
  - Tip (if any)

- **Payment Information:**
  - Payment method(s)

  - Transaction reference (if applicable)

- **Footer:**
  - Thank you message
  - Business policies (if any)

## Troubleshooting

### Problem: Receipt Won't Print

**Possible causes:**
- Printer is off
- Printer has no paper
- Printer not connected
- Wrong printer selected
- Printer driver issue

**Solutions:**
1. **Check printer:**
   - Is it on?
   - Does it have paper?
   - Is it connected (USB/Network)?

2. **Check printer selection:**
   - Go to Settings → Printer
   - Verify correct printer is selected
   - Test print

3. **Try manual print:**
   - Click "Print Receipt" button
   - Select printer
   - Try printing

4. **If still not working:**
   - Contact manager
   - Contact IT support
   - Can email receipt as backup

### Problem: Receipt Prints But Is Blank

**Possible causes:**
- Printer issue
- Paper issue
- Driver problem

**Solutions:**
1. Check printer has paper
2. Check printer is working (test print from Windows)
3. Try printing again
4. Contact IT support

### Problem: Receipt Prints But Is Cut Off

**Possible causes:**
- Wrong paper size
- Printer settings wrong
- Format issue

**Solutions:**
1. Check paper size in settings
2. Verify printer settings
3. Check receipt format
4. Contact manager

### Problem: Receipt Prints Slowly

**Possible causes:**
- Large receipt
- Printer speed
- System performance

**Solutions:**
1. Wait for receipt to finish
2. Check printer is working properly
3. Verify system performance
4. Usually not a problem - just wait

## Email Receipts (If Enabled)

### Sending Email Receipt

**If email feature is enabled:**

1. **After payment**, click "Email Receipt"
2. **Enter customer email address**
3. **Click "Send"**
4. **Receipt is emailed** to customer

**What customer receives:**
- PDF receipt via email
- Same information as printed receipt
- Can be saved or printed later

**Benefits:**
- No paper needed
- Customer has digital copy
- Can be forwarded
- Environmentally friendly

## Receipt Settings

### Printer Selection

**To select printer:**

1. Go to Settings
2. Click "Printer Settings"
3. Select printer from list
4. Test print
5. Save settings

### Receipt Format

**Receipt formats:**
- **58mm** - Narrow thermal paper
- **80mm** - Wide thermal paper
- **Standard** - Regular paper

**To change format:**
1. Go to Settings
2. Click "Receipt Settings"
3. Select format
4. Save settings

### ✅ Do's

- Always print receipts for customers

### ❌ Don'ts

- Don't ignore printer issues
