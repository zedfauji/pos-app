## Script Format

Each script includes:
- **Video title and duration**
- **Step-by-step script** with timing
- **Visual cues** (what to show)
- **Annotations** (optional highlights)
- **Recording notes** (before/during/after)
- **Common mistakes** to avoid
- **Duration breakdown**

---

## Recording Tips

### Before Recording
- Read the script completely
- Practice the steps
- Set up your system
- Test your microphone
- Close unnecessary apps

### During Recording
- Speak slowly and clearly
- Pause between steps
- Follow the script
- Don't rush
- If you make a mistake, pause and correct it

### After Recording
- Watch the video
- Edit mistakes if needed
- Add annotations (optional)
- Export in 1080p MP4
- Upload to hosting platform
