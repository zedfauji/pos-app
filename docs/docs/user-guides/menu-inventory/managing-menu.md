# Managing Menu Items (Managers)

This guide explains how managers manage menu items, prices, and availability.

---

## Overview

**As a manager, you can:**
- Add new menu items
- Update prices
- Mark items as available/unavailable
- Create combos
- Manage modifiers
- View menu history

---

## Adding a Menu Item

### Step-by-Step

1. **Click "Menu Management"** in navigation
2. **Click "Add Item"** button
3. **Enter item details:**
   - **Name:** Item name (e.g., "Margherita Pizza")
   - **Description:** What it is (optional)
   - **Category:** Food, Drinks, Snacks, etc.
   - **Price:** How much it costs (e.g., 12.99)
   - **SKU:** Item code/barcode (optional)
4. **Link to inventory** (if applicable):
   - Select inventory item
   - System checks stock automatically
5. **Set availability:**
   - **Available:** Shows in menu
   - **Unavailable:** Hidden from menu
6. **Add modifiers** (optional):
   - Size options
   - Toppings
   - Customizations
7. **Click "Save"**

**What happens:**
- Item appears in menu immediately
- Servers can add it to orders
- Item is linked to inventory (if configured)

---

## Updating Prices

### Step-by-Step

1. **Click "Menu Management"**
2. **Find the item** to update
3. **Click "Edit"**
4. **Change the price**
5. **Enter reason** (optional but recommended):
   - "Price increase"
   - "Promotion"
   - "Cost adjustment"
6. **Click "Save"**

**What happens:**
- New price takes effect immediately
- Old price is saved in history
- All new orders use new price
- Existing orders keep old price

**Price History:**
- System tracks all price changes
- View history in item details
- Useful for reporting and audits

---

## Marking Items Out of Stock

### Step-by-Step

1. **Click "Menu Management"**
2. **Find the item**
3. **Click "Edit"**
4. **Toggle "Available" to "No"**
5. **Click "Save"**

**What happens:**
- Item disappears from menu
- Servers cannot add item
- Item shows as "Out of Stock" if servers try
- Can mark available again when stock arrives

**Alternative:**
- System automatically marks items out of stock when inventory reaches zero
- You can override manually

---

## Creating Combos

### Step-by-Step

1. **Click "Menu Management"**
2. **Click "Combos" tab**
3. **Click "Add Combo"**
4. **Enter combo details:**
   - **Name:** Combo name (e.g., "Burger Combo")
   - **Price:** Combo price
   - **Description:** What's included
5. **Add items to combo:**
   - Select items from menu
   - Add quantities for each
6. **Set availability**
7. **Click "Save"**

**Benefits:**
- Encourages upselling
- Simplifies ordering
- Can offer discounts
- Tracks combo sales separately

---

## Managing Modifiers

### What are Modifiers?

**Modifiers are options for items:**
- Size: Small, Medium, Large
- Toppings: Add cheese, remove onions
- Customizations: Extra sauce, no pickles

### Adding Modifiers

1. **Click "Menu Management"**
2. **Click "Modifiers" tab**
3. **Click "Add Modifier"**
4. **Enter modifier details:**
   - **Name:** Modifier name (e.g., "Size")
   - **Type:** Single choice or multiple choice
5. **Add options:**
   - Small (+$0.00)
   - Medium (+$1.00)
   - Large (+$2.00)
6. **Link to menu items**
7. **Click "Save"**

## Best Practices

### ✅ Do's

- Keep menu updated
- Update prices promptly
- Mark items out of stock immediately
- Review menu regularly
- Track price changes

### ❌ Don'ts

- Leave outdated items in menu
- Forget to update prices
- Ignore inventory links
- Make changes without testing
