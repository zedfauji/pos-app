# Checking Inventory (Managers)

This guide explains how managers check inventory levels and stock status.

---

## Overview

**As a manager, you can:**
- View current stock levels
- Check low stock alerts
- See inventory status
- Review inventory reports
- Monitor stock turnover

---

## Viewing Inventory

### Step-by-Step

1. **Click "Inventory"** in navigation
2. **View inventory list:**
   - Item name
   - Current quantity
   - Reorder threshold
   - Status (In Stock, Low Stock, Out of Stock)

**Inventory list shows:**
- All items in inventory
- Current stock levels
- Status indicators
- Quick actions

---

## Understanding Stock Status

### Status Indicators

**In Stock (Green):**
- Quantity above reorder threshold
- Item is available
- No action needed

**Low Stock (Yellow):**
- Quantity below reorder threshold
- Item needs reordering soon
- Action recommended

**Out of Stock (Red):**
- Quantity is zero
- Item is unavailable
- Immediate action needed

---

## Low Stock Alerts

### Viewing Alerts

**On Dashboard:**
- Low stock alerts appear at top
- Shows items that need attention
- Click to see details

**In Inventory Page:**
- Filter by "Low Stock"
- See all items below threshold
- Review what needs reordering

### What to Do

**When you see low stock alert:**
1. Check current quantity
2. Review reorder threshold
3. Calculate how much to order
4. Place order with vendor
5. Update stock when delivery arrives

---

## Inventory Reports

### Viewing Reports

1. **Click "Reports"**
2. **Click "Inventory Reports"**
3. **View:**
   - Current stock levels
   - Low stock items
   - Items sold (quantity)
   - Inventory value
   - Turnover rates
   - Reorder recommendations

### Inventory Metrics

**Stock Value:**
- Total value of inventory
- Helps with financial planning

**Turnover Rate:**
- How fast items sell
- Helps with ordering decisions

**Days of Stock:**
- How long current stock will last
- Based on sales rate
