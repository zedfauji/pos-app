## How to Navigate

### Clicking Menu Items

**To go to a page:**
1. Look at the left menu
2. Find the item you want
3. Click on it
4. Page loads

- Click "Tables" â†’ Goes to Tables page

### Using the Dashboard

**The Dashboard is your home base:**
- Shows all tables
- Quick access to everything
- Always available

**To return to Dashboard:**
- Click "Dashboard" in menu
- Or click logo/header
- Always takes you home

## Page Structure

### Most Pages Have:

**Top Section:**
- Page title
- Action buttons (Add, Edit, etc.)
- Search/filter options

**Main Content:**
- Data or information
- Lists, tables, or forms
- Interactive elements

**Bottom Section (Sometimes):**
- Pagination (if many items)
- Summary information
- Action buttons

---

## Common Navigation Patterns

### Going Back

**To go back to previous page:**
- Click "Cancel" button
- Click previous menu item
- Click "Dashboard" to go home
- Use browser back (if applicable)

### Finding Things

**If you can't find something:**
- Check the menu
- Use search (if available)
- Ask your manager
- Check this guide

### Getting Lost

**If you're not sure where you are:**
- Look at the menu (current page is highlighted)
- Check the page title at top
- Click "Dashboard" to start over
