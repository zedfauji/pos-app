## Understanding Table Cards

### Available Table (Green)

**What you'll see:**
- Table number
- "Available" status
- "Start Session" button
- Green background/border

**What you can do:**
- Click to start a session
- View table details
- Assign table to customers

### Occupied Table (Red)

**What you'll see:**
- Table number
- "Occupied" status
- Server name
- Duration (e.g., "45 minutes")
- Current total (e.g., "$32.50")
- "Stop Session" or "View Order" button
- Red background/border

**What you can do:**
- Click to view order
- Add more items
- Stop session
- Process payment

## Common Buttons and Their Meanings

### Table Actions

**Start Session:**
- Appears on available tables
- Click to begin tracking customers at this table
- Starts the timer

**Stop Session:**
- Appears on occupied tables
- Click when customers are ready to pay
- Stops the timer and generates bill

**Add Items:**
- Appears on occupied tables
- Click to add items to the order
- Opens menu selection

**View Order:**
- Appears on occupied tables
- Click to see current order details
- Shows items, quantities, prices

### Payment Buttons

**Process Payment:**
- Appears after stopping a session
- Click to process customer payment
- Opens payment screen

**Split Payment:**
- Appears in payment screen
- Click to split bill among multiple payment methods
- Allows multiple payments for one bill

**Apply Discount:**
- Appears in payment screen
- Click to apply discount to bill
- Requires manager approval (for servers)

**Print Receipt:**
- Appears after payment
- Click to print receipt again
- Also prints automatically

### General Buttons

**Save:**
- Saves changes you made
- Appears in edit screens
- Always click after making changes

**Cancel:**
- Cancels current action
- Discards changes
- Returns to previous screen

**Delete/Remove:**
- Removes an item
- Usually requires confirmation
- Use carefully

**Edit:**
- Allows you to modify something
- Opens edit screen
- Make changes and save

## Menu Selection Screen

**When you click "Add Items":**

**You'll see:**
- **Categories:** Food, Drinks, Snacks, Combos, etc.
- **Search Bar:** Type to search for items
- **Item Cards:** Each item shows:
  - Item name
  - Price
  - Description (if available)
  - Availability status
  - "Add" button

**How to use:**
1. Browse categories or search
2. Click on an item
3. Enter quantity
4. Add modifiers (size, toppings, etc.)
5. Click "Add to Order"
6. Item is added to cart
7. Click "Confirm Order" when done

---

## Payment Screen

**When you click "Process Payment":**

**You'll see:**
- **Bill Summary:**
  - Time cost
  - Items list with prices
  - Subtotal
  - Discount (if any)
  - Tax (if applicable)
  - **Total Amount**

- **Payment Options:**
  - Cash
  - Card
  - Mobile Payment
  - Digital Wallet
  - Split Payment

- **Payment Fields:**
  - Amount input
  - Tip input (optional)
  - Discount input (if approved)

- **Action Buttons:**
  - Process Payment
  - Cancel
  - Apply Discount
  - Split Payment

## Navigation Tips

### Using the Menu

**To navigate:**
- Click on any menu item to go to that page
- Current page is highlighted
- Some pages have sub-menus

### Going Back

**To go back:**
- Click "Cancel" button
- Click previous menu item
- Use browser back button (if applicable)
- Click "Dashboard" to return to main screen

### Finding Things

**If you can't find something:**
- Check the menu
- Use search (if available)
- Ask your manager
- Check this guide
