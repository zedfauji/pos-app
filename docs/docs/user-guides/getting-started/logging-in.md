# Logging In to MagiDesk POS

This guide explains how to log into the MagiDesk POS system.

---

## First Time Login

### Step 1: Open the Application

1. **Find the MagiDesk icon** on your computer desktop
2. **Double-click** to open the application
3. **Wait for application to load** (may take 10-20 seconds)

**If you can't find the icon:**
- Ask your manager for help
- Check the Start Menu
- Contact IT support

### Step 2: Enter Your Credentials

**You'll see a login screen with:**
- Username field
- Password field
- Login button

**Enter your information:**
1. **Username:** Enter the username provided by your manager
   - Example: `john.smith` or `server01`
   - Username is case-sensitive
2. **Password:** Enter the temporary password provided by your manager
   - Example: `TempPass123!`
   - Password is case-sensitive

### Step 3: First Login Password Change

**On your first login, the system will ask you to change your password:**

1. **Enter your current password** (the temporary one)
2. **Enter your new password:**
   - Must be at least 8 characters
   - Should include letters and numbers
   - Can include symbols for extra security
3. **Confirm your new password** (enter it again)
4. **Click "Save"**

**Password Tips:**
- Choose something you'll remember
- Don't use obvious passwords (like "password123")
- Don't share your password with anyone
- Write it down securely if you might forget

### Step 4: You're In!

**After logging in, you'll see:**
- Main dashboard
- Table layout
- Navigation menu
- Your name in the top right

**If you see an error:**
- Check your username and password
- Make sure Caps Lock is off
- Try typing password in notepad first, then copy/paste
- Contact your manager if still having issues

---

## Regular Login (After First Time)

### Daily Login Process

1. **Open MagiDesk application**
2. **Enter your username**
3. **Enter your password**
4. **Click "Login"**

**That's it!** You're logged in and ready to work.

---

## Troubleshooting Login Issues

### Problem: "Invalid Username or Password"

**Possible causes:**
- Typo in username or password
- Caps Lock is on
- Password was changed and you're using old password
- Account was deactivated

**Solutions:**
1. Check username spelling
2. Check Caps Lock is off
3. Try typing password in notepad, then copy/paste
4. Ask manager to reset password
5. Verify account is active

### Problem: "Account is Locked"

**Possible causes:**
- Too many failed login attempts
- Account was deactivated by manager

**Solutions:**
1. Wait 15 minutes and try again
2. Contact manager to unlock account
3. Verify account is still active

### Problem: "System Unavailable"

**Possible causes:**
- System is down for maintenance
- Network connection issues
- Server problems

**Solutions:**
1. Check internet connection
2. Wait a few minutes and try again
3. Contact IT support
4. Check if other staff can log in

### Problem: Application Won't Open

**Possible causes:**
- Application needs to be updated
- Computer needs to be restarted
- Application is corrupted

**Solutions:**
1. Restart your computer
2. Try opening again
3. Contact IT support
4. Ask manager for help

## Security Tips

### Password Security

✅ **Do:**
- Use a strong password (8+ characters, mix of letters/numbers)
- Change password regularly
- Keep password private
- Log out when done

❌ **Don't:**
- Share your password
- Write password where others can see it
- Use obvious passwords
- Leave system logged in

### Account Security

✅ **Do:**
- Log out at end of shift
- Don't let others use your account
- Report suspicious activity
- Keep your account secure

❌ **Don't:**
- Share your login
- Leave system logged in unattended
- Use someone else's account
- Ignore security warnings
